# Actividad DIAct1_1

## Como ejecutar
1. Abrir el proyecto DIAct1_2 en NetBeans
2. Ejecutar la clase Principal.java
3. Se abrira un formulario para introducir los datos de los alumnos
4. Se pueden añadir hasta 5 alumnos. Si intentas añadir mas, aparecera un mensaje de error
5. La etiqueta "Alumno mas joven" se actualiza automaticamente a medida que se agregan alumnos
